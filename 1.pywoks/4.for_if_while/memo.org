
- pythonで繰り返し処理は以下のどちらか
  - for
  - while

- for
  - 英語で「(目的地に)向かって」の意味
  - リストなどを先頭の要素から順に処理していくときに使用
  - 

- while
  - 英語で「〜の間ずっと」の意味
  - 一定の条件が成立する間はずっと処理を行い続ける
  - 条件が成立しなくなると処理の繰り返しが終わる

- if
  - 英語で「もし〜ならば」という意味
  - 条件によってその後の処理を分けるときに使用

- エラー処理
  - pythonには、プログラムの中でエラーが発生したときにその後の処理を切り替える仕組みがある
